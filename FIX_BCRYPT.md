# Исправление проблемы с bcrypt

## Проблема
Ошибка совместимости между passlib и bcrypt. Перешли на прямое использование bcrypt.

## Что изменено:
1. Удален passlib, используется bcrypt напрямую
2. Обновлен requirements.txt
3. Исправлены функции хеширования и проверки паролей

## Что нужно сделать:

### 1. Пересоберите backend контейнер

```powershell
docker-compose build backend --no-cache
```

### 2. Перезапустите backend

```powershell
docker-compose restart backend
```

### 3. Попробуйте зарегистрироваться снова

Обновите страницу регистрации и попробуйте снова.
