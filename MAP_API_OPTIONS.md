# 🗺️ Варианты API для карт и геокодирования

## ✅ Можно использовать любой API!

Текущая реализация использует Yandex Maps, но можно легко переключиться на любой другой API.

## 📊 Сравнение популярных API

### 1. **Yandex Maps** (текущий выбор)
- ✅ **Плюсы:**
  - Отличное покрытие России и СНГ
  - Бесплатно: 25,000 запросов/день
  - Хорошая документация на русском
  - Быстрое геокодирование для российских адресов
- ❌ **Минусы:**
  - Меньше покрытие за пределами СНГ
  - Требует регистрацию в Яндексе

### 2. **Google Maps API**
- ✅ **Плюсы:**
  - Лучшее покрытие по всему миру
  - Отличное качество карт
  - Много функций (Places API, Geocoding API)
- ❌ **Минусы:**
  - Платно (но есть бесплатный кредит $200/месяц)
  - Сложнее настройка
  - Требует привязку карты

### 3. **OpenStreetMap (OSM) + Nominatim** (БЕСПЛАТНО!)
- ✅ **Плюсы:**
  - **Полностью бесплатно** и без ограничений
  - Открытый исходный код
  - Хорошее покрытие
  - Не требует API ключ
- ❌ **Минусы:**
  - Медленнее, чем коммерческие API
  - Меньше функций
  - Требует собственный сервер для production

### 4. **2GIS API**
- ✅ **Плюсы:**
  - Отличное покрытие России
  - Бесплатно для небольших проектов
  - Хорошая детализация адресов
- ❌ **Минусы:**
  - Только Россия и СНГ
  - Ограниченная документация

### 5. **Mapbox**
- ✅ **Плюсы:**
  - Красивые карты
  - Хорошая производительность
  - Бесплатно: 50,000 запросов/месяц
- ❌ **Минусы:**
  - Платно после лимита
  - Требует регистрацию

## 🔄 Как переключиться на другой API

### Вариант 1: OpenStreetMap (БЕСПЛАТНО, без API ключа!)

Самый простой вариант - использовать OpenStreetMap через Nominatim API.

**Преимущества:**
- ✅ Не нужен API ключ
- ✅ Полностью бесплатно
- ✅ Работает сразу

**Недостатки:**
- ⚠️ Медленнее, чем Yandex/Google
- ⚠️ Нет интерактивной карты (нужно использовать Leaflet)

### Вариант 2: Google Maps API

**Преимущества:**
- ✅ Лучшее качество
- ✅ Отличное покрытие

**Недостатки:**
- ❌ Требует API ключ
- ❌ Платно (но есть бесплатный кредит)

### Вариант 3: 2GIS API

**Преимущества:**
- ✅ Хорошо для России
- ✅ Бесплатно для небольших проектов

**Недостатки:**
- ❌ Только Россия/СНГ

## 🚀 Рекомендации

### Для России/СНГ:
1. **Yandex Maps** - лучший выбор (текущий)
2. **2GIS** - альтернатива
3. **OpenStreetMap** - если не нужен API ключ

### Для международного проекта:
1. **Google Maps** - лучший выбор
2. **Mapbox** - альтернатива
3. **OpenStreetMap** - бесплатный вариант

### Для экономии:
1. **OpenStreetMap** - полностью бесплатно
2. **Yandex Maps** - 25,000 запросов/день бесплатно
3. **Mapbox** - 50,000 запросов/месяц бесплатно

## 💡 Что нужно изменить в коде

Если хотите переключиться на другой API, нужно изменить:

1. `frontend/lib/yandex-maps.ts` - заменить функции геокодирования
2. `frontend/components/address-selector.tsx` - заменить загрузку карты
3. `frontend/next.config.js` - изменить переменные окружения
4. `.env` - добавить новый API ключ (если нужен)

## 🎯 Текущая ситуация

Сейчас код использует Yandex Maps, но:
- ✅ Работает без API ключа (fallback режим)
- ✅ Показывает предложения даже без ключа
- ⚠️ Без ключа нет координат (геокодирование не работает)

## 📝 Рекомендация

**Если вы в России/СНГ:**
- Оставьте Yandex Maps (текущий выбор)
- Получите бесплатный API ключ (25,000 запросов/день)

**Если нужен бесплатный вариант без ключа:**
- Можно переключиться на OpenStreetMap
- Потребуется изменить код

**Если нужен международный проект:**
- Переключитесь на Google Maps
- Потребуется изменить код и получить API ключ
