## 🎯 Особенности

### Градиенты
```css
gradient-primary: #667eea → #764ba2
gradient-secondary: #f093fb → #f5576c
gradient-success: #4facfe → #00f2fe
gradient-warm: #fa709a → #fee140
```

### Анимации
- `animate-fade-in` - плавное появление
- `animate-slide-up` - появление снизу
- `animate-scale-in` - масштабирование
- Задержки для последовательных анимаций

### Утилиты
- `text-gradient` - градиентный текст
- `glass` - glass morphism эффект
- Кастомные скроллбары

## 📱 Адаптивность

- 📱 Мобильные устройства (320px+)

## 🚀 Использование

### Toast уведомления
```typescript
import { useToast } from '@/hooks/use-toast'

const { success, error, warning, info } = useToast()

success('Операция выполнена!', { title: 'Успех' })
error('Произошла ошибка', { title: 'Ошибка' })
```

### Skeleton загрузка
```typescript
import { Skeleton, SkeletonCard, SkeletonList } from '@/components/ui/skeleton'

<Skeleton className="h-4 w-3/4" />
<SkeletonCard />
<SkeletonList count={5} />
```

### Progress бары
```typescript
<Progress value={75} variant="success" showLabel />
```

## 🎨 Цветовая палитра

- **Primary**: Синий градиент (основной)
- **Success**: Зеленый градиент
- **Warning**: Желтый
- **Error**: Красный
- **Info**: Голубой
