# Настройка Yandex Maps API

## Получение API ключа

1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите в аккаунт Яндекс
3. Создайте новый проект или выберите существующий
4. Перейдите в раздел "JavaScript API и HTTP Геокодер"
5. Создайте новый ключ API
6. Скопируйте ключ

## Настройка в проекте

### 1. Добавьте ключ в .env файл

Создайте файл `.env` в корне проекта (если его нет) и добавьте:

```env
NEXT_PUBLIC_YANDEX_MAPS_API_KEY=ваш-api-ключ-здесь
```

### 2. Перезапустите frontend

```powershell
docker-compose restart frontend
```

## Возможности

- ✅ Автодополнение адресов при вводе
- ✅ Выбор адреса на интерактивной карте
- ✅ Перетаскивание маркера на карте
- ✅ Обратное геокодирование (координаты → адрес)
- ✅ Валидация адреса перед созданием заказа

## Использование

1. Введите адрес в поле ввода - появятся подсказки
2. Или нажмите "На карте" для выбора на карте
3. Кликните на карте или перетащите маркер
4. Адрес автоматически заполнится

## Ограничения бесплатного тарифа

- До 25,000 запросов в день
- До 1,000,000 запросов в месяц

Для production рекомендуется:
- Кэширование результатов геокодирования
- Ограничение частоты запросов
- Использование платного тарифа при необходимости
