# 🗺️ Настройка карт для выбора адреса

## ✅ Что реализовано

1. **Компонент выбора адреса** с автодополнением
2. **Интерактивная карта** Yandex Maps для выбора адреса
3. **Геокодирование** - преобразование адреса в координаты
4. **Обратное геокодирование** - координаты в адрес
5. **Валидация адреса** перед созданием заказа
6. **Улучшенный UI/UX** на уровне премиум-сервисов

## 🚀 Быстрый старт

### 1. Получите API ключ Yandex Maps

1. Перейдите на https://developer.tech.yandex.ru/
2. Войдите в аккаунт Яндекс
3. Создайте новый проект
4. Перейдите в раздел "JavaScript API и HTTP Геокодер"
5. Создайте новый ключ API
6. Скопируйте ключ

### 2. Добавьте ключ в .env

Создайте или откройте файл `.env` в корне проекта и добавьте:

```env
NEXT_PUBLIC_YANDEX_MAPS_API_KEY=ваш-api-ключ-здесь
```

### 3. Пересоберите и перезапустите frontend

```powershell
docker-compose build frontend --no-cache
docker-compose restart frontend
```

### 4. Готово!

Откройте http://localhost:3000/orders/new и попробуйте:
- Ввести адрес - появятся подсказки
- Нажать "На карте" - откроется интерактивная карта
- Кликнуть на карте - адрес автоматически заполнится

## 🎨 Возможности

### Автодополнение адресов
- Вводите адрес и получайте подсказки в реальном времени
- Выбирайте из предложенных вариантов

### Интерактивная карта
- Кликните на карте для выбора адреса
- Перетаскивайте маркер для точной настройки
- Автоматическое определение адреса по координатам

### Валидация
- Проверка наличия координат перед созданием заказа
- Предупреждение, если адрес не выбран на карте

## 📋 Технические детали

- Используется Yandex Maps JavaScript API 2.1
- Геокодирование через Yandex Geocoder API
- Компонент `AddressSelector` для переиспользования
- Утилиты в `lib/yandex-maps.ts`

## ⚠️ Важно

- Без API ключа карта не будет работать
- Бесплатный тариф: 25,000 запросов/день
- Для production рекомендуется кэширование результатов

## 🔧 Устранение проблем

### Карта не загружается
1. Проверьте, что API ключ добавлен в `.env`
2. Проверьте консоль браузера на ошибки
3. Убедитесь, что ключ активен в личном кабинете Yandex

### Подсказки не появляются
1. Проверьте подключение к интернету
2. Убедитесь, что API ключ имеет доступ к Geocoder API
3. Проверьте лимиты запросов в личном кабинете
