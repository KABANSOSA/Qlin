# Исправление синтаксической ошибки

## Проблема
Ошибка компиляции в `frontend/app/orders/[id]/page.tsx` на строке 73.

## Решение

### 1. Перезапустите frontend

```powershell
docker-compose restart frontend
```

Подождите 30 секунд, пока frontend перекомпилируется.

### 2. Очистите кэш Next.js (если ошибка сохраняется)

```powershell
docker-compose exec frontend rm -rf .next
docker-compose restart frontend
```

### 3. Проверьте файл

Файл должен иметь правильную структуру:
- Функция `OrderDetailPageContent` начинается на строке 12
- `return (` на строке 72
- Внешний `<div>` на строке 73
- Внутренний `<div>` на строке 74
- `<Card>` на строке 119
- Закрывающие теги на строках 232-234

## Если не помогает

Проверьте, что все скобки и теги закрыты правильно.
