## 📋 Что нужно сделать для применения

### 1. Применить миграцию БД

```powershell
docker-compose exec backend alembic upgrade head
```

### 2. Перезапустить сервисы

```powershell
docker-compose restart backend frontend
```

### 3. Проверить работу

- Откройте http://localhost:3000
- Зарегистрируйтесь
- Создайте заказ
- Проверьте профиль
